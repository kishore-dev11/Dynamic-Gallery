# 🌍 Dynamic Image Gallery (React)

This project is a **QuadraCycle** built using React JS.  
It displays images based on categories and allows users to filter and search images dynamically.

---

## 🚀 Features

- 📸 Display images dynamically using React
- 🔄 Reusable Image Card component
- 🔍 Search functionality
- 🗂️ Category-based filtering (Bloom, Sunblaze, Autumn, Frostbite)
- 📱 Responsive design (Mobile & Desktop)
- ⚡ Fast and clean UI using CSS Grid
- ♻️ Component-based architecture

---

## 🧩 Tech Stack

- React JS
- JavaScript (ES6)
- HTML5
- CSS3
